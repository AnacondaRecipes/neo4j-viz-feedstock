Installing neo4j-viz
==================

Installing `neo4j-viz` from the main channel can be achieved by:

```
conda install neo4j-viz
```

It is possible to list all of the versions of `neo4j-viz` available on your platform with `conda`:

```
conda search neo4j-viz
```
